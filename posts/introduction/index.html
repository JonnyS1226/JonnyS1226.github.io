Introduction to ML

September 2, 2020 One-minute read
Machine Learning
machine learning

  • model or function set
    • Linear Model
    • Non-linear Model
      • deep-learning
      • neural networks
      • SVM, K-NN, Decision tree, e.t.c
  • Supervised Learning（监督学习）: label, input, output
    • regression（回归）
    • classification（分类）
      • binary classification
      • multi-class classification
  • semi-supervised Learning（半监督学习）: partly labeled data, partly unlabeled data
  • Unsupervised Learning（非监督学习）: unlabeled data
  • Structured Learning（结构化学习）: like GAN, get a structured object
  • other frontier
    • anomaly detection（异常检测）: how can AI know that it don’t know
    • transfer learning（迁移学习）: what can irrelevant data contribute to the results
    • meta learning（元机器学习）: learn to learn
    • life-long learning（持续学习）: continual learning
    • Reinforcement Learning（强化学习） : learning from critics
    • explainable AI（可解释性）: reason for AI results
    • adversarial attack（对抗攻击）: how to address malicious attack to AI
    • network compression（网络剪枝）: how to prune large model
