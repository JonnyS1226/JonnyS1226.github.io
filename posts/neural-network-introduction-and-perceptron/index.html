2. 感知器算法

  • 多层神经网络，区分几个概念：输入层，输出层，隐藏层(hidden layer)

  • 感知器算法是两层神经网络（输入层$x_1,x_2$, 系数$w_1,w_2$, 偏置b, 输出层$y$）

  • activation function 取阶跃函数

  • 感知器算法能实现逻辑与、或、非运算

    • $\bigcap$ $w_1=w_2=1,b=-2 \rightarrow y=\varphi(x_1+x_2-2)$; only if $x_1=x_2=1$, $y=1$
    • $\bigcup$ $w_1=w_2=1,b=-0.5\rightarrow y=\varphi(x_1+x_2-0.5)$; only if $x_1=1 $ OR $x_2=1,y=1$
    • $～$ $w_1=-0.6,w_2=0,b=0.5\rightarrow y=\varphi(-0.6x_1+0.6)$; only if $x_1=1,y=0$; only if $x_1=0,y=1$

  • 感知器算法可以区分与或非，然而无法对异或问题进行分类

  • 即对于非线性可分的问题（生活中大多数都是非线性可分问题），就要使用多层功能神经元，且同样要保证激活函数是非线性的，本文第3点将简单证明使用三层神经网络能模拟任意决策面
