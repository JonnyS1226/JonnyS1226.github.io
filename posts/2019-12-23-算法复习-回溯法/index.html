算法简介

回溯法，与分支限界法，都属于树的搜索技术，回溯法就是一个在约束条件下对解空间树进行深度优先搜索，并且在搜索的过程中剪枝：当搜索到某个结点，发现当前路径不满足条件时，回溯至祖先结点重新开始下一条路径的搜索。

典型例题

1. n皇后问题

1) 题目描述

在一个n*n的棋盘上放置n个皇后，任意两个皇后不能处在同一行、同一列、同一斜线。 对于用户输入的n，输出n皇后问题的解数。

2) 分析

  • 回溯法

    • 画出解空间树，实质就是从左向右，从上至下，顺序依次摆放皇后，每次摆放后都判断一下是否满足条件
    • 若不满足条件，撤销这次摆放，将皇后摆放到下一个位置

  • 位运算法

3) 代码

    private static int methodNum = 0;
	public static void solveNQueenBackTrack(int row, int[][] chessBoard){
        if (row == chessBoard.length)
        {
            methodNum++;
            System.out.println("第" + methodNum + "种：");
            for (int i = 0; i < chessBoard.length; i++) {
                for (int j = 0; j < chessBoard[0].length; j++) {
                    System.out.print(chessBoard[i][j] + " ");
                }
                System.out.println();
            }
            return;
        }
        for (int j = 0; j < chessBoard[0].length; j++) {
            // 如果采用二维数组的方式，一定要在摆放之前清空该列的之前的摆放
            for (int k = 0; k < chessBoard[0].length; k++)
            {
                chessBoard[row][k] = 0;
            }
            chessBoard[row][j] = 1;
            if (isSafe(chessBoard, row, j))
            {
                solveNQueenBackTrack(row + 1, chessBoard);
            }
        }
    }

    public static boolean isSafe(int[][] chessBorad, int row, int col){
        int interval = 1;
        while (row - interval >= 0)
        {
            //下面是行、列、左上、右上四种情况，&操作符两端应用了短路，不能改变顺序，否则会造成数组越界
            //采用一维数组更方便，可以直接省去同一列的情况
            if ((chessBorad[row - interval][col] == 1) 
                ||
                (col - interval >= 0 && chessBorad[row][col - interval] == 1) 
                ||
                (col + interval < chessBorad.length && 
                 chessBorad[row - interval][col + interval] == 1) 
                ||
                (col - interval >= 0 && chessBorad[row - interval][col - interval] == 1))
            {

                return false;
            }
            interval++;
        }
        return true;
    }

4) 复杂度分析

根据解空间树，可知对于n*n棋盘的总的种类：$T_1(n) = 1+n+n^2+n^3+…+n^n \leq 2n^n$。

对于isSafe函数，时间复杂度取决于row值，故：$T_2(n)=1+2+…+n=\frac{n(n+1)}{2}$。

故总时间复杂度：$T(n) = T_1(n) \times T_2(n)=O(n^{n+2})$，由于过程中有剪枝，最终会远小于这个值。

2. 0-1背包问题

1) 题目描述

已知有n件物品，物品i的重量为Wi，价值为Pi。背包最大承重为m。 求能使背包所装物品价值总和最大的物品装包方法。

2) 分析

背包问题是一个经典问题，有多种解决方法，包括动态规划，回溯法，分支限界法，贪心算法等，此处着重说回溯法。

  • 回溯法：

    • 先将物品放入，然后判断是否满足重量要求，若不满足则再拿出。

    • 用这种回溯方法得到某种满足要求的放物品方法，并且记录其总价值，然后最后选取出最大价值的那种放置物品方法。

3) 代码

package backTrack;
import java.util.Scanner;

/**
 * 01背包问题
 */
public class knapsack_problem {
    private static int currentMaxValue = 0;     //最大价值
    private static int currentValue = 0;        //当前价值
    private static int currentWeight = 0;       //当前重量
    private static int n;                       //物品数
    private static int[] currentrs;             //当前装物品结果
    private static int[] rs;                    //最终装物品结果
    private static int remainValue;             //当前剩余物品价值总和
    //i 为解空间树的深度
    public static void knapsackBackTrack(int i, int[] weight, int[] value, int totalWeight){
        if (i == n)
        {
            if (currentValue > currentMaxValue)
            {
                currentMaxValue = currentValue;
                for (int j = 0; j < n; j++) {
                    rs[j] = currentrs[j];
                }
            }
            return;
        }
//         *         下面本质是完整的回溯模型，只不过增加了剪枝的操作
//         *         currentrs[i] = 1;
//         *         currentWeight += weight[i];
//         *         currentValue += value[i];
//         *         knapsackBackTrack(i + 1, weight, value, totalWeight);
//         *         currentrs[i] = 0;
//         *         currentWeight -= weight[i];
//         *         currentValue -= value[i];
//         *         knapsackBackTrack(i + 1, weight, value, totalWeight);
        //对第i层，即第i个物品进行处理，先搜索左子树，即放入的情况
        remainValue -= value[i];
        if (currentWeight + weight[i] <= totalWeight)
        {
            currentrs[i] = 1;
            currentWeight += weight[i];
            currentValue += value[i];
            knapsackBackTrack(i + 1, weight, value, totalWeight);
            currentWeight -= weight[i];
            currentValue -= value[i];
        }
        //以下这步用于剪枝，将那些必然不会满足最大价值的装物品情况剪枝
        if (remainValue + currentValue > currentMaxValue)
        {
            currentrs[i] = 0;
            knapsackBackTrack(i + 1, weight, value, totalWeight);
        }
        remainValue += value[i];
    }

    public static void main(String[] args) {
        System.out.println("输入总承重量：");
        Scanner sc = new Scanner(System.in);
        int totalWeight = sc.nextInt();
        System.out.println("输入物品数：");
        n = sc.nextInt();
        System.out.println("依次输入各个物品的重量和价值");
        int[] weight = new int[n];
        int[] value = new int[n];
        for (int i = 0; i < n; i++) {
            weight[i] = sc.nextInt();
            value[i] = sc.nextInt();
            remainValue += value[i];
        }
        rs = new int[n];
        currentrs = new int[n];
        for (int i = 0; i < n; i++) {
            rs[i] = 0;
            currentrs[i] = 0;
        }
        knapsackBackTrack(0, weight, value, totalWeight);
        System.out.println("结果是：");
        for (int i = 0; i < n; i++) {
            System.out.print(rs[i] + "  ");
        }
    }
}

4) 复杂度分析

根据解空间树可以得到，最坏情况下$T(n)=O(2^n)$，其中$n$为解空间深度，也即物品数。

而由于有剪枝，实际时间复杂度会小于$O(2^n)$。
